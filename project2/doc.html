Documentation for project 2

Overview & Goals

For Project 2, I will be creating a search feature for the Art Institute of Chicago's public API. I'm sure it already exists on their website itself, but I just want to create a clean interface that feels easy to use. I will use CSS grid or flexbox to arrange artwork results in rows of three or four. Each artwork will be displayed as equally-sized squares with some spacing. Hovering over them will display information about the image and allow the user to click on the image, with in-window 'pop-up'-like dialogues if the user clicks on the description.
