Documentation for project 2

Overview & Goals

For Project 2, I will be creating a search feature for the Art Institute of Chicago's public API. I'm sure it already exists on their website itself, but I just want to create a clean interface that feels easy to use. I will use CSS grid or flexbox to arrange artwork results in rows of three. Each artwork will be displayed as equally-sized squares with some spacing. Hovering over them will display information about the image and allow the user to click on the image, with in-window 'pop-up'-like dialogues if the user clicks on the description. If I end up having extra time at the end of the project, I will create a "favorites" feature in which the user can "star" artworks that they like and create a custom collection, stored in local storage.

Purpose

The purpose of this project is to create a streamlined search tool to find art from the Art Institute of Chicago. You can find artworks on their website, but it is not always easy to find what you're looking for. Is the tool revolutionary? No. But it will provide a nicer interface (in my opinion) for the user to interact with.

Audience

There is no target audience, really. It's for those who appreciate art and want to be able to explore what the Art Institute of Chicago has to offer. For all ages and eyes. For those who aren't blind.
